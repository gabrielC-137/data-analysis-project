This folder contains the original raw datasets used in the project.

The dataset used in this project was obtained from the Kaggle (https://www.kaggle.com/datasets/shreyasur965/recent-earthquakes), a repository widely used for the development of data science and machine learning projects. This dataset provides detailed information on 1,137 earthquakes from around the world. It includes a wide range of attributes such as magnitude, location details, time, and various seismological measurements. The data was collected using the EveryEarthquake API from RapidAPI, offering researchers, seismologists, and data enthusiasts a rich resource for analysis, machine learning projects, and geological studies.

No modifications were applied to the files stored in this directory.
